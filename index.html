Project Details

More about the Projects

Ernst & Young

Data collection using web scraping - creation of reusable scripts.

Data acquisition through web scraping – automating Jira ticket data transfer in Excel.

As part of my student project in collaboration with Ernst & Young (EY) and HTW Berlin, I developed a Python script that automates the manual process of extracting data from Jira tickets. The goal was to create a reusable script that extracts relevant ticket data directly from Jira and accurately transfers it to an Excel file.

Main objectives of the project:
- Reduction of manual workload through automation
- Minimize transmission errors and improve data quality
- Flexibility through a variable API key to use the script with different Jira accesses
- Ensuring the completeness of the data by recording all relevant ticket fields
- Easy handling and reusability of the script for future projects

The script not only optimizes the process, but also ensures more efficient data processing and relieves employees of repetitive tasks. In the long term, it contributes to improving internal processes at EY and better utilizing resources.

KPMG

Development of an EAM tool in the context of data and integration architecture using LeanIX as an example

Within the scope of the project, the example company GreenTech and the order-to-cash process were specified as the content and process context.

Objectives of the project:
- Development of a metamodel to map the data architecture
- Definition of the interface and integration architecture
- Expansion of the demo space in LeanIX

All defined project goals were successfully achieved within the planned time period.

IT service center

Implementing a booking portal using SharePoint.

The task was to digitize the previously analog and labor-intensive booking process for healthcare services at ITDZ Berlin and to develop an automated appointment booking portal using SharePoint, Nintex Forms and Nintex Workflows.

Challenges:
- Replacing the time-consuming and error-prone manual booking process.
- Structuring of SharePoint lists for simultaneous use by employees and the BGM team.
- Implementation of complex form logic for role-based display and editing of content.
- Implementation of automated workflows for appointment calculation, seat display and email notification.
